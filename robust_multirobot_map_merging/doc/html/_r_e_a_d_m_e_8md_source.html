1 # Robust Multi-Robot Map Merging
2 
3 This repository contains an implementation of the robust map merging method presented in
4 ```
5 J. G. Mangelson, D. Dominic, R. M. Eustice, and R. Vasudevan,
6 “Pairwise Consistent Measurement Set Maximization for Robust Multi-robot Map Merging.”, ICRA 2018.
7 ```
8 
9 The **purpose of this package** is to find a subset of inliers in a large set of inter-robot loop closures. Due to large measurement errors and perceptual aliasing in multi-robot applications, we need a **robust method** to select which measurements to trust.
